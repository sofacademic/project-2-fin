# Project Title
Caterpillar Convention Website

## Project Description
This is a mock-up design for a website dedicated to a caterpillar-based convention.

## Intended User or Player
This interface is designed for someone looking to quickly find information regarding this event.

## User / Player Goal
The user should be able to quickly locate on the screen the information they are looking for.

## Inspiration Interfaces
I was inspired by the Goblin Market and the Clearwater Marine Aquarium's websites. Both provided ideas for a layout based on simple and clear sections and basic geometric shapes that make it easy to focus on the information rather than how it is displayed.

## Design Choices
- Layout: I tried to make the layout simple while still being visually interesting. I used simple box shapes to create sections on the page and put two images along with them.
- Color: I used a green color palette to go along with the caterpillar theme and match the images and to not be too intense or dramatic.
- Typography: I used a simple rounded font to be easily readable, soft on the eyes, and to go along with the caterpillar theme.
- Visual hierarchy: I put the website logo at the top so it is clear what the website is, and immediately after that there is the information regarding the next event and its location, date, and time, as that is likely the most important information to the user. The next two sections deal with any other questions the user may have and have appropriately sized headings.
- Feedback states: The contacts at the bottom of the screen change color when hovered over to show that they are interactive.
- Responsive design: The size and layout of the main section of the page can change sizes depending on the screen to better accomodate different devices.
